# Domain Docs

guidotto.dev is a single-context portfolio site.

- Notion remains canonical for profile, portfolio, project, task, and personal knowledge.
- Use narrow Notion lookups for public-safe source material when content facts may matter.
- Do not commit raw personal information dumps or broad Notion exports.
- Use `CONTEXT.md` only if resolved site/domain language needs to be captured locally.
- Use `docs/adr/` only for hard-to-reverse decisions.

When writing issue titles, content changes, docs, or refactor proposals, preserve the terms and public-safe boundaries from Notion and already-public sources.

If a change contradicts the source-material policy, say so explicitly before proceeding.
